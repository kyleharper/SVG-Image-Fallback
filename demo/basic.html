Basic Example

This example shows a basic implementation. Things to note;

  • "data-name" is always required to be set as a data attribrute.
  • "data-location" can be set either on the element with a data attribrute or in the plugin options as shown below.
            
// It's always a good idea to set your base image location 
// in the options like so;
$('.js-svgCheck').SVGImgFallback({
    "location" : "../your/img/directory/"
});

// Then you can add your image tag with the image name going 
// into the "data-name" attribrute. The SVG format will be added 
// via JS and if SVG isn't supported it'll fallback to PNG unless 
// told otherwise.
<img data-name="image-name" alt="alt text here"/>
