SVG to JPG Examples

Site Wide Example

This example shows how you change the fallback format from "png" to "jpg" for all required images. This would be very rare as normally when working with SVG graphics you'll require a PNG fallback for the transpancy. Please note;

  • The "jpg" format has been added in the options below.
            
$('.js-svgCheck').SVGImgFallback({
    "format" : "jpg",
    "location" : "../your/img/directory/"
});

Individual Element

This example shows how you can change the fallback manually from ".png" to ".jpg" for a individual item. Please note;

  • The "data-format" attribrute is set on the one image where you'd want a jpg fallback to be used.
  • The other image hasn't got the "data-format" attribrute set so it uses the default format of "png".
