Multiple Locations Example

There may be times where you'll have images located in multiple locations and this plugin allows you to specify different locations using the "data-location" attribrute. Please note;

  • Your default image path (the one used the most in your project) should be referenced in the plugin options like below.
        
// Default (most used) image location set in the options
$('.js-svgCheck').SVGImgFallback({
    "location" : "../your/img/directory/"
});

// Individually per item for seperate locations
<img data-location="images/sub-folder/" data-name="image-name"/>
  • One the default is set you can use the "data-location" attribrute like the example below.

"data-location" added. This will be detected in the plugin and used to craft the image src.

"data-location" not set. This will use either the default location "images/" or the location set in the plugin options.
